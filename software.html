CPTRA

CPTRA is a software package for cross platform transcriptome analysis (CPTRA). It is in particular useful for annotated cDNA library and digital gene expression analysis. For CPTRA package, please click here. To cite CPTRA, please click here.

HDXanalyzer

HDX analyzer is an RPY package for statistical analysis of HDX mass spectrometry data toward quantitive analysis of protein structure dynamics. To download HDXanalyzer, please click here. For more information about the package, please click here.

SAES

SARS (Statistical Analysis of Real-time pcr Software) is an RPY package for statistical analysis of real-time PCR, implementing the previous algorithms published in our lab. The SARS package is still under peer review. Once it is accepted for publication, it will be available.
